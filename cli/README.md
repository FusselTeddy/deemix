## Usage

- `./deemix-cli -h` - Display help
- `./deemix-cli <url>` - Download url

### Parameters
| Parameter                        | Description                          | Default        |
| -------------------------------- | ------------------------------------ | ------------   |
| `-p /path/to/config`             | Path to the config folder            | deemix default |
| `-b bitrate`                     | Bitrate to download                  | `MP3_320`      |
